# Классы и объекты

## Модификаторы доступа

```python
self.name = "Alexey" # Публичный (Public)
self._age = 27 #  Защищенный (Protected)
self.__city = "Moscow" # Приватный (Private)
```

### Публичный / Public

```python
self.name = "Alexey" # Публичный (Public)
```

### Защищенный / Protected

```python
self._age = 27 #  Защищенный (Protected)
```

### Приватный / Private

```python
self.__city = "Moscow" # Приватный (Private)
```
